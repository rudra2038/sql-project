# SQL Project: Employee & Department Management System

## Overview
This is a simple SQL project that manages employees and departments using **MySQL**.  
It includes table creation, data insertion, and several useful queries to analyze employee data —
like finding top salaries, averages, and employees above the company average.

---

## Database Design
**Tables:**
- **Department:** dept_id, dept_name  
- **Employee:** emp_id, emp_name, dept_id, salary, join_date  

Each employee belongs to one department (linked by a foreign key).

---

## What’s Included
 Creating and linking tables  
 Inserting sample data  
 Using `JOIN`, `GROUP BY`, and `SUBQUERY`  
 Creating a `VIEW` for quick reports  
 Writing a small `STORED PROCEDURE` for salary raise
